# DENTAL — Landing Page

Landing page para el consultorio odontológico "DENTAL", diseñada para 
centralizar la presencia digital del negocio. Permite a los pacientes 
potenciales consultar el catálogo de tratamientos, precios y testimonios, 
así como agendar citas en línea, eliminando la dependencia del contacto 
presencial o por mensajería instantánea.
